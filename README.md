# 🎵 Really Cool Vinyl - Widget Canvas & Music Player

Welcome to **Really Cool Vinyl**! An interactive experience designed to unwind, enjoy, and explore your favorite music in a visually unique environment.

🔗 **Live demo:** https://vinyl-player-beta.vercel.app/

---

## 🌟 The Concept

Have you ever felt the frustration of wanting to listen to a song that is **only on YouTube** and having to deal with the platform's intrusive interface just for that?

**Really Cool Vinyl** solves this elegantly:
1. **Upload your files**: Bring those tracks you've downloaded from YouTube.
2. **Persistent Playlist**: Your songs are stored locally in your browser (IndexedDB), so your playlist will be there the next time you open the app.
3. **Background Listening**: Perfect for having music in the background while you work, study, or just relax.

## 🎨 Interactive Widgets

The application is more than just a player; it's a fully customizable **Widget Canvas** (on desktop):

- **💿 Vinyl Disc**: Watch it spin as the music plays.
- **📊 Track Info**: Visual details of the current track.
- **🎛️ Premium Controls**: A clean interface to manage your session.
- **📜 Dynamic Playlist**: Easily manage, reorder, and clear your track list.
- **🏗️ Drag & Drop**: Drag and place widgets anywhere on your screen to create your ideal musical workspace.

## 🚀 Technologies

Built with a modern stack to ensure smoothness and aesthetics:

- **React 19** + **Vite**: Blazing fast development and execution.
- **Tailwind CSS 4**: Modern, responsive styling.
- **Framer Motion**: Smooth animations and premium micro-interactions.
- **Lucide React**: Clean and minimal iconography.
- **IndexedDB**: Fast and secure local data persistence.

## 🛠️ Installation and Setup

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/vinyl-player.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```

## 🎮 Playing with Widgets

1. **Desktop Experience**: Use the bottom toolbar to toggle widgets. Click and drag them across your entire screen!
2. **Mobile Experience**: A compact, optimized experience to take your music anywhere.
3. **Interactive Upload**: Use the upload widget (top-left corner) to add your favorite MP3s.
